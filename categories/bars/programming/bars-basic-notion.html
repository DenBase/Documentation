БАРС СВОДЫ

Основные понятия

В Дизайнере можно открывать для редактирования одновременно более одной формы. Каждая форма открывается в отдельной закладке, которая добавляется в верхней части рабочей области

Метаструктура

Короткое описание

Раздел «Метаструктура» предназначен для создания и редактирования структурных компонентов формы. Метаструктура(или структура) формы – описание схемы хранения данных отчетной формы. Структура отчетной формы состоит из свободных ячеек и таблиц.

Увязки

Увязки – контрольные соотношения показателей формы, соблюдение которых обеспечивает сбалансированность и непротиворечивость данных формы. Раздел «Увязки» предназначен для создания и редактирования увязок формы. Увязки помогают сотрудникам МИАЦ проводить экспертизу форм, а пользователям ЛПУ – контролировать ошибки заполнения формы.

Макросы

Раздел «Макросы» предназначен для работы с макросами.

Экранная форма

Раздел «Экранная форма» предназначен для загрузки шаблона экранной формы. Экранная форма определяет внешний вид отчетной формы (в пределах возможностей системы, разумеется). У каждой отчетной формы может быть только одна экранная форма (ЭФ), в отличие от печатной формы, которых может быть несколько.

Печатные формы

Раздел «Печатные формы» предназначен для создания и редактирования печатных форм. Печатная форма – это выгрузка данных формы в файл Excel, который обычно используется для предоставления отчетности по этой форме на бумажном носителе. Внешний вид печатной формы (ПФ) может значительно отличаться от экранной формы, кроме того, сами данные можно структурно представить в другом виде, либо поместить в ПФ лишь часть данных.

Дополнительно

Раздел «Дополнительно» предназначен для создания и редактирования бланка абонентского пункта. Бланк абонентского пункта никогда не придется создавать самостоятельно, т.к. подобный процесс вообще не описан ни в какой документации, а создаваемые по умолчанию бланки вполне хороши. Однако следует знать, что это такое. Выгрузка абонентского пункта – выгрузка данных формы в файл Excel, специально размеченный так, что данные из него можно загрузить обратно в БАРС. Таким образом, если у пользователя проблемы с подключением к Интернет, или введенные им данные по какой-то причине не сохраняются, ему рекомендуют заполнить форму в выгрузке абон.пункта. Эта выгрузка является практически полным аналогом экранной формы.


Все вышеперечисленные компоненты отчетной формы хранятся в наборе файлов, который называется пакетом файлов метаописания формы. Чтобы открыть папку формы и получить доступ к ним, следует нажать правой кнопкой мыши по закладке с названием формы и выбрать пункт Открыть папку формы.
